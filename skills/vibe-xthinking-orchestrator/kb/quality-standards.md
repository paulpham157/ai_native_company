# Quality Standards — vibe-xthinking-orchestrator

> **Quality criteria cho mọi analysis output.**

## Universal Standards

| Criterion | Description | Pass/Fail |
|-----------|-------------|-----------|
| Evidence-backed | Mọi claim phải có evidence source | Fail nếu unsupported claims |
| Structured | Output theo đúng schema | Fail nếu missing required fields |
| Clear | Language rõ ràng, không ambiguous | Fail nếu vague/unclear |
| Actionable | Insights phải actionable | Fail nếu purely descriptive |
| Confidence | Score phản ánh evidence quality | Fail nếu score không consistent |

## MODE TOPIC Standards

- Research phase phải có ≥ 3 sources
- Mỗi agent output phải có evidence tracking
- Gaps phải được identify với severity
- Aggregate confidence phải consistent với từng phase

## MODE PROBLEM Standards

- Root causes phải traceable đến evidence
- Level hierarchy phải logical (1 → 2 → 3...)
- Recommendations phải address root causes
- Framework selection phải justified

## MODE DECISION Standards

- Criteria phải được define trước khi score
- Mỗi score phải có justification
- Sensitivity analysis khuyến khích
- Recommendation phải có reasoning chain
